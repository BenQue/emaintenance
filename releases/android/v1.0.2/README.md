## 🐛 已知问题及解决方案

### 1. 应用图标不显示
- **问题**：某些 Android 启动器可能不显示应用图标
- **解决方案**：
  - 设置 → 应用 → E-Maintenance → 添加到桌面
  - 或通过应用抽屉查找并启动

### 2. 摄像头权限问题
- **问题**：扫码功能需要摄像头权限
- **解决方案**：
  - 设置 → 应用 → E-Maintenance → 权限 → 摄像头（允许）

### 3. 网络连接问题
- **问题**：无法连接到服务器
- **解决方案**：
  - 确保设备与服务器在同一网络
  - 检查服务器地址配置是否正确
